# El-Clima

Aplicacion Web que muestra los datos climaticos (temperatura actual, minima. y maxima, humedad, viento, etc.) del dia de hoy y de los proximos 5 dias.

[Abrir Web](https://franco-navarro.github.io/El-Clima/).
